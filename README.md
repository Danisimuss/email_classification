# Email Classification — автоматическая сортировка писем по категориям

Скрипт на Python для анализа и автоматической сортировки входящих писем. Программа сканирует указанную папку, анализирует тему и тело каждого письма на наличие ключевых слов и перемещает их в соответствующие тематические папки.

## 📋 Категории сортировки

| Категория | Описание | Примеры ключевых слов |
|-----------|----------|----------------------|
| **Черновик** | Черновики и неотправленные письма | черновик, draft, не отправлено |
| **Спам** | Нежелательные рассылки и реклама | спам, акция, выиграли, казино |
| **Важное** | Критическая информация, требующая внимания | важно, срочно, urgent, critical |
| **Уведомления** | Автоматические оповещения от сервисов | уведомление, подтверждение, notification |
| **Работа** | Рабочая переписка, задачи, отчёты | проект, отчёт, встреча, дедлайн |
| **Ошибка файла** | Письма с проблемами вложений | ошибка файла, повреждён, corrupted |
| **Несортированное** | Письма, не подошедшие ни под одну категорию | (категория по умолчанию) |
